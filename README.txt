Задание C#:
Каталог с текстовыми файлами с расширением .txt, в каждом из которых записаны от 100 до 1000 целых чисел, каждое на новой строке.
Написать консольную утилиту, которая:

1. Считает из консольной строки путь к каталогу, в котором находятся эти текстовые файлы
2. По этому пути прочтёт содержимое всех файлы и составит список из тех чисел, которые:
	2.1. Являются уникальными
	2.2. Имеют остаток от деления на 4 равным 3
3. Отсортирует получившиеся числа в порядке убывания
4. Запишет результат в тот же каталог в файл result.txt, каждое число на новой строке

Бонус:
Можно написать отдельно функцию для генерации исходных данных.

Реализация
Консольная утилита реализована в файле "Program.cs", где пользователю предоставляется выбрать одно из действий: 
1 - создать текстовые файлы,  2 - прочитать и обработать текстовые файлы, 3 - завершить программу.
При нажатии 1 программа запустит функцию по созданию текстовых файлов. Функция потребует ввода пути к каталогу, где будут созданы файлы и ввода количества создаваемых файлов. Файлы будут иметь имена "file_i.txt" (где i - это номер файла)
При нажатии 2 программа запустит функцию по чтению и обработке текстовых файлов в каталоге, который укажет пользователь, а после сохранит всё в файл "result.txt".
При нажатии 3 будет произведен выход из программы

Запуск
1. Установить среду выполнения .NET (Microsoft .NET Framework или .NET Core).
2. Скачайте проект из этого репозитория на свой компьютер
3. Через среду выполнения запустите файл "Program.cs"
